ABOUT

This is a project about teaching LLM to understand the context of pdf files. We are trying
to make it understand the context of the pdf files and make it answer the questions
about the pdf files. You can upload the pdf files and ask questions about the pdf files. We will
try to answer the questions about the pdf files. this application is made using the python and
flask. We are using the LangChain for the understanding the context of the pdf files.

PROCESS

  • Upload the pdf file.
  • Select the vector store
  • Ask the question about the pdf file.
  • Wait for the answer.
